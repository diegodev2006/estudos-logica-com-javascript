## Lógica de Programação com JavaScript
# Resumo

Descrição: Repositório com exercícios e exemplos práticos para aprender os conceitos fundamentais de lógica de programação usando JavaScript.
Público-alvo: Iniciantes e autodidatas que querem entender raciocínio lógico, estruturas de controle e funções aplicadas em JavaScript.
Abordagem: Explicações curtas, trechos de código comentados e exercícios práticos organizados por tema.
O que você vai aprender

Fundamentos: Tipos de dados, variáveis e operadores.
Condicionais: if, else if, else, switch.
Estruturas de repetição: for, while, do...while.
Funções: Declaração, parâmetros, retorno e escopo.
Objetos e JSON: Criação, leitura e manipulação básica.
Boas práticas: Organização de código, nomes significativos e pequenos exercícios guiados.
Estrutura do repositório

Curso/ — pastas por tópicos com exemplos e exercícios.
novos codigos/ — exercícios e experimentos adicionais.
README.md — este resumo e instruções rápidas.
Como usar

Abra os arquivos .js com o seu editor de código.
Para executar um exemplo, instale o Node.js (se necessário) e rode:

`node caminho/para/o/arquivo.js`
